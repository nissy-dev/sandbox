# プロダクトアイデア

作りたい・試したいプロダクトのアイデアをためる場所。

## 使い方

1. `_template.md` をコピーして、ファイル名は `YYYY-MM-DD-短いタイトル.md` などにする
2. 思いついたことを書き足す（完璧に書かなくてよい）
3. 着手したら sandbox 直下にプロジェクト用ディレクトリを切る

## ステータス

| 記号 | 意味 |
|------|------|
| 💡 | アイデア段階 |
| 🔍 | 調査・検証中 |
| 🚧 | 実装中 |
| ✅ | 完了・棚上げ |
